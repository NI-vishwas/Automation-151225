### Core Scripts

#### Calculations
- **`01-simple_interest.py`** - Calculate simple interest with hardcoded values
- **`02-simple_interest.py`** - Interactive simple interest calculator with user input
- **`03-numsum.py`** - Calculate sum and average of 10 numbers from user
- **`04-numsum.py`** - Calculate sum and average of dynamic number of entries (until user enters -999)

#### String & Text Processing
- **`05-username_extracter.py`** - Extract username from email ID (before @ symbol)
- **`06-extract_num.py`** - Extract numbers from a string and calculate their sum
- **`07-extract_punctuations.py`** - Extract and display punctuations from a string
- **`08-summary_generator.py`** - Generate file summaries (lines, words, characters) and save to summary.log

#### Demo/Learning Examples
- **`demo_conditionals.py`** - Demonstrates if, elif, else conditional statements
- **`demo_loops.py`** - Demonstrates while loop with increment/decrement
- **`demo_string.py`** - String manipulation and indexing examples
- **`demo_function1.py`** - Function definitions, parameters, global variables
- **`demo_calsses.py`** - Object-oriented programming with inheritance and method overriding

#### File Handling
- **`file_handling.py`** - File I/O operations with exception handling and context managers
- **`save_data.py`** - Demonstrates importing custom utilities and saving data
